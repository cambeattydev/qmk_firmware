# ErgoDox EZ Configuration, slowly getting better

This layout has 3 layers:$
1. Base layers
2. Rider shortcuts, F-keys, and number pad
3. Mouse movement and clicks

This keymap will change a lot, so the readme maybe out of date. Check the keymap file to be sure

Need to add the ` key. Turns out I need it
